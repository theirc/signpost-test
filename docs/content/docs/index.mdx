# Welcome to Signpost AI

Signpost AI is a comprehensive framework for building intelligent AI agents and workflows. It provides a visual flow-based interface for creating complex AI systems that can handle text, audio, images, and documents.

## What is Signpost AI?

Signpost AI is a **visual AI workflow builder** that allows developers and non-developers alike to create sophisticated AI agents without writing complex code. Think of it as "Lego blocks for AI" - you can drag and drop different components (called "workers") to build powerful AI systems.
